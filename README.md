# 🔍 Accordion project

. A project built to learn how to create an accordion.

--- 

## 🚀 Website

🔗 Access: https://izaqueqv.github.io/accordion/

---

This project was developed with the goal of learning how to build an accordion, exploring concepts such as:

- Semantic HTML structuring, using the relationship between `input` and `label`  
- Creating an accordion using only CSS with the `:checked` pseudo-class  
- State manipulation with JavaScript through the `checked` property  
- Event handling with `addEventListener`  
- Using `preventDefault()` to override the browser's default behavior  
- Layout organization with **Flexbox**  
- Element positioning with `position: relative` and `absolute`  
- Visibility control using `max-height` and `overflow: hidden`  
- Smooth animations with `transition`  
- Applying CSS best practices, such as reset and `box-sizing` standardization  


---

## 🛠 Technologies  used 

- HTML5  
- CSS3  
- JavaScript
